## Supported Operations

### Substraction

The calculator now supports subtraction between two numeric values.

- Users can input a first value
- Choose the subtraction operation (-)
- The result corresponds to the difference between the two numbers

This functionality is handled by a specific JavaScript function dedicated to subtraction logic.

### Addition

The calculator currently supports basic addition between two numeric values.

- Users can enter two numbers
- Select the addition operation (+)
- Click the calculate button to display the result

This feature is implemented using a dedicated JavaScript function to ensure clear separation between calculation logic and user interface handling.
